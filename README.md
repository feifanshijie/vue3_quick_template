# vue3_quick_template

> vue3 快速开发模板，集成了常用组件

### 快速使用
```bash
git clone git@github.com:feifanshijie/vue3_quick_template.git
cd vue3_quick_template
npm install
npm run dev
```

### 目录结构
```
|-src
    |-apis          api请求
    |-assets        静态资源
    |-components    组件
    |-config        配置
    |-language      国际化
    |-pages         view
    |-public        根目录
    |-router        路由
    |-store         状态
    |-styles        样式
    |-utils         工具封装
```

### 相关组件
```bash
# vue-router 没什么好说的
npm install --save-dev vue-router
# 国际化
npm install --save-dev vue-i18n
# 数据共享
npm install --save-dev vuex 
# vue支持
npm install --save-dev @vitejs/plugin-vue
# jsx支持
npm install --save-dev @vitejs/plugin-vue-jsx
# 数据请求
npm install --save-dev axios
# vite.config.ts path提示报错需要安装
npm install --save-dev @types/node
```
